# Excel 模板文件说明

## 📁 模板文件位置

根据项目设计，Excel 模板文件应放置在 `templates/` 目录下。

**注意**: 由于当前是代码仓库，实际的 `.xlsx` 文件需要通过以下方式获取：

### 方式 1: 使用测试数据生成器（推荐）

运行测试数据生成脚本会自动生成符合规范的 Excel 文件：

```bash
cd scripts
npm install
npm run generate
```

生成的文件位于 `scripts/test_data/`：
- `orders_test_data.xlsx` - 可作为订单模板参考
- `verification_records_test_data.xlsx` - 可作为核销记录模板参考
- `cash_flow_records_test_data.xlsx` - 可作为收银流水模板参考

### 方式 2: 手动创建模板

参考 [doc/EXCEL_TEMPLATES_GUIDE.md](./doc/EXCEL_TEMPLATES_GUIDE.md) 中的字段规范，在 Excel 中创建空白模板。

## 🎨 如何创建标准模板

### 步骤 1: 打开 Excel

新建一个空白工作簿。

### 步骤 2: 输入表头

在第 1 行输入对应的字段名（英文）。

### 步骤 3: 设置格式

#### 日期列格式
1. 选中日期列（如 F 列）
2. 右键 → "设置单元格格式"
3. 选择"自定义"
4. 输入格式代码：`yyyy-mm-dd hh:mm:ss`

#### 金额列格式
1. 选中金额列（如 E 列）
2. 右键 → "设置单元格格式"
3. 选择"数值"
4. 小数位数：2

#### 文本列格式
保持默认的"常规"或设置为"文本"。

### 步骤 4: 添加示例数据（可选）

在第 2-3 行添加 1-2 条示例数据，方便用户理解。

### 步骤 5: 冻结首行

1. 选中第 2 行
2. 菜单 → "视图" → "冻结窗格" → "冻结首行"

这样滚动时表头始终可见。

### 步骤 6: 保存文件

1. 文件 → 另存为
2. 文件名：`orders_template.xlsx`（或其他模板名）
3. 文件类型：Excel 工作簿 (*.xlsx)
4. 保存到 `templates/` 目录

### 1. 表头样式
- 背景色：浅蓝色 (#E3F2FD)
- 字体：加粗

### 2. 边框
- 为整个数据区域添加细边框

## ⚠️ 重要提醒

### 必须遵守的规则

1. **第一行必须是表头**
   - 字段名必须与文档中定义的一致
   - 区分大小写（建议全部小写，用下划线分隔）

2. **从第二行开始是数据**
   - 不要有空行
   - 不要有合并单元格

3. **必填字段不能为空**
   - 参考 [EXCEL_TEMPLATES_GUIDE.md](./doc/EXCEL_TEMPLATES_GUIDE.md) 中的必填字段说明

4. **日期格式统一**
   - 必须是 `YYYY-MM-DD HH:mm:ss` 格式
   - 不要在日期中包含文字（如"上午"、"下午"）

5. **金额为纯数字**
   - 不要包含货币符号（¥、$等）
   - 不要包含千分位逗号（1,000.00 ❌ → 1000.00 ✅）

6. **避免特殊字符**
   - 不要包含换行符、制表符
   - 避免使用 emoji

### 常见错误

❌ **错误示例**:
```
订单号,门店,金额,时间
DY001,北京店,¥199.00,2026/7/11 12:30
```

✅ **正确示例**:
```
order_id,store_id,amount,order_time
DY001,STORE_001,199.00,2026-07-11 12:30:00
```

## 🔧 CSV 格式模板

如果需要提供 CSV 格式模板，注意事项：

### 1. 编码格式
- **必须使用 UTF-8 编码**
- 建议在文件开头添加 BOM 头（`\ufeff`），确保 Excel 正确显示中文

### 2. 分隔符
- 使用逗号 `,` 作为分隔符
- 如果字段值包含逗号，需要用双引号包裹

### 3. 换行符
- 使用 Unix 换行符 `\n` 或 Windows 换行符 `\r\n`
- 保持一致即可

### 4. 示例 CSV 内容

```csv
order_id,store_id,store_name,product_name,amount,order_time,status,source
DY20260711001,STORE_001,北京朝阳店,双人套餐A,199.00,2026-07-11 12:30:00,completed,douyin
DY20260711002,STORE_001,北京朝阳店,单人午餐,59.90,2026-07-11 13:15:00,completed,douyin
```

### 5. 保存为 CSV

1. Excel → 文件 → 另存为
2. 文件类型：CSV (UTF-8) (*.csv)
3. 确认编码为 UTF-8

## 🎯 最佳实践

### 给商家的建议

1. **首次使用前**
   - 下载标准模板
   - 查看示例数据
   - 理解每个字段的含义

2. **准备数据时**
   - 从收银系统导出原始数据
   - 复制到标准模板中
   - 调整格式符合要求

3. **导入前检查**
   - 确认必填字段都有值
   - 确认日期格式正确
   - 确认金额为数字
   - 删除空行和无关数据

4. **导入后验证**
   - 查看导入报告
   - 确认成功/失败数量
   - 如有错误，根据提示修正后重新导入
